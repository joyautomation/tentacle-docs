# tentacle-web

SvelteKit frontend for the Tentacle platform.

## Tech Stack

- SvelteKit 2.9
- Svelte 5
- TypeScript 5.6
- Vite 6
- D3 for visualizations

## Port

Development: 5173

## Running

```bash
cd tentacle-web
deno run -A npm:vite dev
```

## Key Features

- Real-time variable monitoring via GraphQL subscriptions
- PLC tag browsing with progress indicator
- MQTT configuration management
- Device/variable management UI
